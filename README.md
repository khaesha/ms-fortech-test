## Getting Started

### Installation


```bash
# Clone the repository
git clone https://github.com/khaesha/ms-fortech-test.git

# Navigate to the project directory
cd ms-fortech-test

# Install dependencies
npm install
```

### Run the project

Running project locally

```bash
npm run dev
```

### Testing

```bash
npm run test
```
